# Seed/Resumes/

Drop in every resume version you'd want JobFlow to consider — your master, any role- or
industry-specific variants, and prior versions you've actually used. Add a cover-letter template too
if you have one.

`seed-resume-library` catalogs these into `Context/resume-and-cover-conventions.md` with "use this one
when…" rules, and `application` later starts from the right variant when tailoring for a posting.

(You can delete this README once you've added your files.)
