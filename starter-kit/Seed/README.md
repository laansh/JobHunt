# Seed/

This is where you **upload material you already have** so the seed prompts can build your
[`../Context/`](../Context/) library. You don't write anything new here — you drop in existing files.

| Folder | Drop in… | Feeds |
|---|---|---|
| `VoiceSamples/` | emails, blog posts, LinkedIn writing, cover letters — anything in your own voice | `seed-voice-and-style` |
| `Resumes/` | every resume version you'd want considered, plus a cover-letter template | `seed-resume-library` |
| `Projects/` | case studies, post-mortems, retro notes, presentations — anything documenting what you did | `seed-project-list` |

**Quality of seeding sets the ceiling.** The voice guide is only as good as the samples; the project
list only as deep as the source docs. Aim for ~20+ writing samples and ~10+ project documents if you can.

Two of the five Context files come from conversation instead of uploads: `about-me` (a free-text dump
you paste into the prompt) and `scoring-rubric` (a guided Q&A). Nothing to upload for those.
