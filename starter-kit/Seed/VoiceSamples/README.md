# Seed/VoiceSamples/

Drop in anything you wrote in your own voice: emails, blog posts, LinkedIn posts/comments, past cover
letters, newsletters, Slack/forum messages you're proud of. More is better — aim for ~20+.

`seed-voice-and-style` reads everything here to learn your tone, rhythm, and word choices so later
drafts sound like you, not like generic AI. Mix in both casual and formal samples if you have them —
the prompt captures multiple registers.

(You can delete this README once you've added your files.)
