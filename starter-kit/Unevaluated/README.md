# Unevaluated/

Your inbox for new postings. Drop a job description here (paste into a `.md` file, or save the PDF)
whenever you spot something interesting — don't stop to evaluate it.

When you're ready, run `jds-eval-light` (or just tell the orchestrator "triage my unevaluated folder").
It scores each posting and, for the ones worth pursuing, creates a proper folder under `../Roles/`.
Postings that don't make the cut can be deleted or moved to `../Archive/`.
