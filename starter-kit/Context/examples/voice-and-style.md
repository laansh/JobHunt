# Voice & Style

## Tone descriptors
- **Direct** — leads with the conclusion, then supports it
- **Concrete** — prefers a number or example over an adjective
- **Warm-but-brief** — friendly opener, then gets to the point
- **Low-ceremony** — minimal hedging, no corporate filler

## Sentence & structure patterns
- Opens emails with the ask or takeaway in line one
- Short paragraphs (1–3 sentences); uses bulleted lists for options
- Closes with a single clear next step, not a sign-off paragraph
- Em-dashes for asides; rarely uses semicolons

## Signature choices
- Uses: "Here's the tradeoff," "Net:", "One ask:", "Worth noting"
- Avoids: "synergy," "circle back," "excited to," exclamation points, "just wanted to"

## Registers
| Register | When to use | What changes |
|---|---|---|
| Standard | most outreach, cover letters at startups | contractions, first person, brief |
| Formal | public-sector / regulated employers, exec audiences | no contractions, fuller sentences, more context up front |

## Dos and don'ts
- Do: state the value proposition first; quantify; keep it to one screen
- Don't: open with "I am writing to"; pad with enthusiasm; bury the ask

## Calibration notes
- Default to Standard. Switch to Formal when the employer is government, finance, healthcare, or when
  writing to a C-level stranger. Keep concision in both — formal ≠ verbose.
